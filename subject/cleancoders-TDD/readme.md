### 룰
- 한 게임은 10개의 프레임이다.
- 각 프레임은 보통 2개의 롤을 갖는다. 즉, 공을 2번 굴릴 수 있다.
- Strike : 첫 번째 롤에서 핀 10개를 쓰러트린 경우. 다음 두 번의 roll의 점수를 추가로 얻는다. 
- Spare : 두 번째 롤에서 핀 10개를 쓰러트린 경우. 다음 1번의 roll의 점수를 추가로 얻는다.
- 마지막 프레임에서는 Strike인 경우 2번, Spare인 경우 1번의 추가 roll을 얻는다.
